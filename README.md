# DNS Cluster

## Cíl projektu

Cílem tohoto projektu je vytvořit autoritativní DNS clustery. Tyto clustery bude sloužit k poskytování služeb DNS (Domain Name System) pro doménu ".local." včetně reverzních dotazů.
